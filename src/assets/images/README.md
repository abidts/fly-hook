# Images Directory

This directory contains the local image assets for the FLY HOOK Tour & Travel website hero slider.

## Required Images

The following images need to be placed in this directory:

1. `dal-lake.jpg` - High-quality image of Dal Lake with houseboats
2. `gulmarg-snow.jpg` - Beautiful snow-covered landscape of Gulmarg
3. `pahalgam-landscape.jpg` - Scenic view of Pahalgam valley
4. `sonmarg-landscape.jpg` - Golden meadows of Sonmarg
5. `kashmir-valley.jpg` - Panoramic view of J&K valley

## Notes

These images will replace the external Unsplash URLs in the HeroSlider component to improve loading performance and reduce dependency on external resources.
